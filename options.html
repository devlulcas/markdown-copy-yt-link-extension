Markdown Copy YT Link

Configure the markdown template used by the YouTube copy button.

  • Variables: {{title}}, {{url}}, {{channel}}, {{thumbnail}}, {{publishedAt}}
  • Formatters: uppercase, lowercase, clip-extra, ignore-parentheses
  • Date patterns for publishedAt: YYYY, YYYY-MM, YYYY-MM-DD, YYYY-MMMM
  • Example: {{publishedAt|YYYY-MMMM|uppercase}}
  • Examples: {{title|clip-extra}}, {{title|ignore-parentheses|uppercase}}
